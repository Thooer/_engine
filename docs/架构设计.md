游戏引擎**常见模块清单**，包括**功能**与**依赖关系**。

---

## 1) Platform 与 OS Abstraction（平台层）

**功能**

* 窗口/消息循环、输入设备枚举（键鼠手柄触控）
* 文件系统路径、权限、线程/定时器、动态库加载
* 高 DPI、剪贴板、IME、崩溃收集（可选）

**主要依赖**

* 依赖：无
* 被依赖：众多

---

## 2) Application / Engine Core（引擎核心）

**功能**

* 引擎生命周期：init → update → shutdown
* 模块加载/卸载、服务定位（Service Locator）或依赖注入
* 主循环、时间源（帧时间/固定步长）、全局配置

**主要依赖**

* 依赖：Platform、Logging/Diagnostics（可选）
* 被依赖：众多

---

## 3) 场景与世界(Scene / World / ECS)

**功能**

* 实体/组件/系统（ECS）、场景图或 Entity 层级、Prefab/Prototype
* 查询、变更检测、系统调度（与 JobSystem 协作）
* 跨场景加载、分区（Zone/Chunk）、运行时实例管理

**主要依赖**

* 依赖：Core、JobSystem（可选）、Reflection/Serialization（常见）、Events
* 被依赖：Gameplay、Physics、Animation、AI、Scripting、Rendering（通过提交流）

---

## 4) 资源与资产管线（Resource/Asset System）

**功能**

* 统一资源 ID/URL、依赖图、引用计数/生命周期、热重载
* 导入器（FBX/GLTF/PNG/WAV…）、构建产物（cooked data）
* Streaming（分块加载）、压缩/打包、版本与缓存

**主要依赖**

* 依赖：FileSystem、Serialization、Reflection（可选但常见）、JobSystem
* 被依赖：Rendering（材质/纹理/网格）、Audio（音频资源）、Animation、UI、Scripting

---

## 5) Rendering（渲染）

通常拆成“图形 API 后端 + 渲染器 + 渲染资源”。

### 5.1 RHI / Gfx Backend（渲染硬件接口）

**功能**

* 封装 Vulkan/D3D12/Metal/OpenGL（创建 device/swapchain/buffer/texture/pso）
* Shader 管理、命令队列、同步、资源屏障

**依赖**

* 依赖：Platform（窗口句柄）、Diagnostics
* 被依赖：Renderer、Tools（GPU 捕捉/调试）

### 5.2 Renderer（渲染器：管线与可见性）

**功能**

* Render Graph / Frame Graph、Pass 管理
* 可见性裁剪（frustum/occlusion）、批处理、LOD
* 光照与阴影、后处理、GI（可选）、粒子渲染（可选）

**依赖**

* 依赖：RHI、Scene（读取渲染组件）、Asset（网格/材质）
* 被依赖：Editor Viewport、Runtime

---

## 6) Animation（动画）

**功能**

* 骨骼动画、Blend Tree、状态机、IK/约束（可选）
* 动画资源采样与压缩、根运动（root motion）
* 与渲染/物理的同步（布娃娃/碰撞体）

**主要依赖**

* 依赖：Asset、Scene/ECS、Math（基础库你说不介绍，但这里默认存在）
* 被依赖：Gameplay、Rendering（蒙皮）、Physics（ragdoll）

---

## 7) Physics（物理）

**功能**

* 刚体/软体（可选）、碰撞检测、约束、角色控制器
* 触发器/射线/扫掠查询、物理材质
* 物理步进与回放、与 ECS 同步

**主要依赖**

* 依赖：Scene/ECS、JobSystem（常见）、DebugDraw（可选）
* 被依赖：Gameplay、AI（导航障碍）、Networking（回滚/同步可选）

---

## 8) Navigation / AI（导航与 AI）

**功能**

* NavMesh 生成/更新、寻路（A\*）、动态障碍
* 行为树/GOAP/Utility AI（任一）
* 感知系统、黑板、群体/队形（可选）

**主要依赖**

* 依赖：Scene/ECS、Physics（常用）、Asset（Nav 数据）
* 被依赖：Gameplay

---

## 9) Audio（音频）

**功能**

* 声音播放/混音、3D 空间音效、RTPC/参数、总线与效果器
* 音频资源流式加载、语音/麦克风（可选）
* 与场景同步（listener/emitter）

**主要依赖**

* 依赖：Platform（音频设备）、Asset、Scene/ECS
* 被依赖：Gameplay、UI

---

## 10) Input（输入）

**功能**

* 设备抽象：键鼠手柄触控陀螺仪
* Action/Mapping 系统、重绑定、输入事件队列
* 文本输入（IME）与 UI 集成

**主要依赖**

* 依赖：Platform
* 被依赖：Gameplay、UI、Editor

---

## 11) UI / GUI（界面）

分运行时 UI 与编辑器 UI（编辑器 UI 通常更重）。

**功能**

* 布局系统、字体/文本、渲染（批处理/九宫格/矢量可选）
* 输入焦点、控件体系、动画/过渡
* 数据绑定（可选）、本地化（可选）

**主要依赖**

* 依赖：Rendering（或单独 UI 渲染器）、Input、Asset（字体/图集）
* 被依赖：Gameplay、Editor

---

## 12) Scripting（脚本）

**功能**

* 脚本运行时（Lua/C#/JS…）、绑定生成（反射或手写）
* 热重载、调试（断点/堆栈）、沙盒权限（可选）
* 脚本组件与系统

**主要依赖**

* 依赖：Scene/ECS、Reflection、Asset（脚本包）
* 被依赖：Gameplay、Tools（调试器）

---

## 13) Gameplay Framework（玩法框架）

**功能**

* 常用游戏对象范式：Actor/Component、能力系统、属性与状态、伤害/触发
* 关卡规则、任务、对话（可选）
* 存档点、检查点、重生等高层逻辑支持

**主要依赖**

* 依赖：Scene/ECS、Input、UI、Audio、Physics、Animation、Scripting（可选）
* 被依赖：具体游戏逻辑（项目层）

---

## 14) Networking / Multiplayer（联网，可选）

**功能**

* 连接与会话、消息与 RPC、序列化与压缩
* 状态同步：快照/插值/预测回滚（高级）
* 权威模型、复制（replication）、反作弊（可选）

**主要依赖**

* 依赖：Scene/ECS（复制对象）、Serialization、Time、Physics（回滚时）
* 被依赖：Gameplay、Tools（网络分析）

---

## 15) Save/Load & Serialization（存档与序列化）

**功能**

* 场景/实体/资源引用序列化
* 版本兼容、差量（patch）、加密/校验（可选）
* 快照与回放（可与网络共用）

**主要依赖**

* 依赖：Reflection（非常常见）、Asset、FileSystem
* 被依赖：Editor、Runtime、Networking

---

## 16) Editor（编辑器）

**功能**

* 场景编辑、资源浏览器、导入设置、Inspector（属性面板）
* Gizmo、Prefab 编辑、Play In Editor
* 工具链：烘焙（lightmap/navmesh）、构建发布、打包

**主要依赖**

* 依赖：几乎全部：Scene、Rendering、Asset、UI、Input、Serialization、Scripting（常见）
* 被依赖：无（顶层应用）

---

## 17) Tooling / Diagnostics（调试与性能工具）

**功能**

* Profiler（CPU/GPU）、Frame capture、统计与可视化
* 日志与断言、崩溃收集、内存/资源泄漏检测
* DebugDraw（物理/导航可视化）

**主要依赖**

* 依赖：Core、Rendering（GPU 工具）、Platform
* 被依赖：Editor、Runtime（开发构建）

---

# 一个“典型依赖层级”（从底到顶）

* **Platform**
* **Core**
* **File/IO（通常在 Platform 或 Core 旁边）**
* **Asset/Serialization/Reflection**
* **Scene/ECS + JobSystem**
* **Subsystems**：Rendering / Physics / Audio / Animation / Input / UI / AI
* **Gameplay Framework / Scripting / Networking**
* **Editor + Tools**
