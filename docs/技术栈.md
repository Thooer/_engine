开发语言Rust
窗口与操作系统接口: winit
跨平台图形接口桥梁: raw-window-handle
游戏输入控制 (手柄/震动): gilrs
图形 RHI: wgpu (强制配置为仅启用 Vulkan / DX12 / Metal 后端，彻底抛弃 OpenGL 状态机)
着色器语言与交叉编译: WGSL + naga (无缝集成于 wgpu，编写一次 WGSL 即可零成本编译为 SPIR-V / HLSL / MSL)
核心数学库：glam
引擎架构：bevy_ecs 
物理引擎：rapier3d 、rapier2d
音频系统：kira
IMGUI：egui(社区有现成的 egui-wgpu 和 egui-winit 绑定库)
资源加载与序列化：gltf、image、ktx2 / ddsfile、serde + bincode + ron
**反射系统（场景序列化 + 编辑器）: bevy_reflect**
**场景序列化：DynamicScene (bevy_reflect)**

**场景序列化技术栈**:
- 场景文件 (.ron): RON 格式 - 人类可读、与 bevy_ecs 兼容
- 引擎配置 (.toml): TOML 格式 - 简洁易读
- 网络/持久化: bincode - 高效二进制
多线程与任务调度：rayon (用于计算密集型) 或 tokio (用于 IO 密集型)，
Profiling：tracing，profiling 搭配 tracy-client
资源热更（Asset Hot-Reloading）：监听 WGSL 着色器、glTF 模型、JSON 配置文件的变化。使用 notify 库监控文件夹。替换 wgpu 显存。
逻辑热更（Scripting）的选择：WASM
VFS: vfs 或 object_store
